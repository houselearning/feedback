# The HouseLearning Feedback Page
## What is this Repository?
This repository is the feedback page for HouseLearning. HouseLearning depends on your feedback to optimize your suggestions for other learners!
## How do I submit feedback?
You can fill out feedback *here*: [HouseLearning Feedback Page](https://feedback.houselearning.org)
<br>
With the feedback page, try not to include any sensitive details like your personal name, address, ssn (*just an example! Please don't. Your response will be deleted right away*)
